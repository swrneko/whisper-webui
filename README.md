---

## Installation
1. clone repository:

```shell
git clone https://github.com/swrneko/whisper-webui.git
cd whisper-webui
```

2. create env:
  - with conda:
  ```shell
  conda create -n whisper-webui-by-swrneko python=3.10  
  conda activate whisper-webui-by-swrneko
  ```

  - with pyenv: 
  ```shell
  python -m venv .venv
  source .venv/bin/activate
  ```

3. run app:
```shell
python app.py
```
